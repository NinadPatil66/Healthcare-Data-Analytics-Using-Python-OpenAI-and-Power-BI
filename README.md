# Project 1:
## Project Description: 
This project automates the processing of dense patient encounter summaries using Large Language Models. The solution transforms natural language transcripts into structured data, focusing on key clinical entities such as demographics, symptoms, diagnosis and treatment plans. It features an automated ICD-10 encoder that matches clinical findings to the global diagnostic standard, optimizing the workflow for insurance processing and medical record management. These medical transcripts can be used for other medical documentation, such as for insurance purposes, but as they are densely packed with medical information, extracting the key data accurately can be challenging. In this project I leveraged the OpenAI API to automatically extract medical information from these transcripts and automate the matching with the appropriate ICD-10 codes. ICD-10 codes are a standardized system used worldwide for diagnosing and billing purposes, such as insurance claims processing.

## Goal:
Automatically extract medical information from medical transcripts such as Age, Medical Specialty, Recommended Treatment and automate the matching with the appropriate ICD-10 codes.
